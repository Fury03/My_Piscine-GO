# Piscine Go

A collection of exercises completed during the 01 Edu Go Piscine at LEARN2EARN NG.

## Quests

| Quest | Description |
|-------|-------------|
| Quest 1 | Shell & CLI Fundamentals |
| Quest 2 | If & Else, For Loops, Variables, PrintRune & ASCII |
| Quest 3 | Strings Manipulation, Range Loop, Pointers, Modulo & Division |
| Quest 4 | Iterativity & Recursivity |
| Quest 5 | Runes, Bytes & Strings |
| Quest 6 | Os.Args |
| Quest 7 | Make & append methods |
| Quest 8 | Structures in Go |
| Quest 9 | Functions as Arguments |
| Quest 10 | Hackathon (A series of different tasks to complete) |
| Quest 11 | Introduction to LInked Lists |
| Quest 12 | Introduction to Binary Trees |


## Tech Stack
- Language: Go,Shell.
